## Features
- Modern, responsive portfolio built with **Next.js 14 (App Router)** + **Tailwind CSS**
- Beautiful dark/light mode toggle
- Real professional photo & animated hero section
- All **11+ real projects** from academic, internship & PFE
- Stunning project cards with hover effects
- Skills, certifications, experience & achievements from CV
- Download CV button (English PDF)
- Fully mobile-optimized

## Tech Stack
React • Next.js • Tailwind CSS • TypeScript • Vercel
